# Contributing to Live QR Code

Thank you for your interest in contributing to Live QR Code! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/liveqrcode.git`
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)

## Development

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **QR Engine**: qr-code-styling
- **Language**: TypeScript

## How to Contribute

### Reporting Bugs
- Open an issue on [GitHub Issues](https://github.com/Latracal-Solutions/liveqrcode/issues)
- Include steps to reproduce, expected behavior, and screenshots if applicable

### Suggesting Features
- Open a feature request issue
- Describe the feature and why it would be useful

### Submitting Pull Requests
1. Create a new branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Test thoroughly on mobile and desktop
4. Commit with a clear message: `git commit -m "Add: your feature description"`
5. Push and open a PR against `main`

### Code Style
- Use TypeScript for all new files
- Follow existing code patterns and naming conventions
- Use Tailwind CSS for styling (no inline styles)
- Ensure mobile responsiveness
